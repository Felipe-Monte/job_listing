# Projeto Job Listing
- Projeto para listar progamadores do github, guardando eles no seu localstorage
- Buscar usuários API github

#### Funcionalidades

- Salvar usuários no localstorage
- Consular usuários do github
- Apagar todos os usuários
- Consultar API
